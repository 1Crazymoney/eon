EON
Realtime Dashboard Framework

Realtime Charts, Graphs, and Maps.

Build your realtime dashboard with EON. Stream data into a variety of maps and charts. Take over the world.

Get Started

Realtime Charts.

Powered by C3.

Include scripts and styles.

<script type="text/javascript" src="http://pubnub.github.io/eon/lib/eon.js"></script>
<link type="text/css" rel="stylesheet" href="http://pubnub.github.io/eon/lib/eon.css" />

Define chart.

eon.chart({
  channel: 'eon-chart',
  limit: 20,
  history: true,
  generate: {
    bindto: '#body-spline'
  }
});

Publish data.

PUBNUB.publish({
  channel:  'eon-chart',
  message:  });

Realtime Maps.

Powered by Mapbox.

Include scripts and styles.

<script type="text/javascript" src="http://pubnub.github.io/eon/lib/eon.js"></script>
<link type="text/css" rel="stylesheet" href="lib/eon.css" />

Define map.

eon.map({
  channel: 'eon-map',
  id: 'html-id',
  mb_token: '...',
  mb_id: 'ianjennings.l896mh2e'
});

Publish data.

PUBNUB.publish({
  channel:  'eon-map',
  message:  });
